A simple party system for BungeeCord. It allows basic functionality, such as creating and disbanding parties, inviting players, warping, etc. I tried to make this plugin for my personal server, but I decided to publish it for this portfolio, since it's a more decent project.

Randomize (Bukkit)

A rudimentary block/mob drop randomizer for Spigot/Bukkit. Whenever you kill a mob or break a block, it drops a random item. It doesn't use loot tables. Each block will have its own drop, for example, if you break a dirt block and it drops a diamond helmet, it will drop a diamond helmet every time you break a dirt block.
